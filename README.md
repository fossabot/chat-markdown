# ChatMarkdown

A ChatGPT client built with remark and CodeMirror 6, optimized for markdown.

## Features

- [x] GFM support
- [x] Code block support
- [x] Code highlight support
- [x] LaTeX support
- [ ] Link support
- [ ] Task list support
- [ ] Mermaid support
- [ ] Diagram support
- [ ] Image support
- [ ] Profile support
- [ ] i18n support

## Demo

[chat-markdown.vercel.app](https://chat-markdown.vercel.app)

## Development

### Install

```bash
pnpm install
```

### Dev

```bash
pnpm dev
```

### Build

```bash
pnpm build
```
